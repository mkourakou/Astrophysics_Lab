## Requirements
Τα προγράμματα είναι γραμμένα σε python 3 σε jupyter notebook.
Βασικά πακέτα που χρησιμοποιούνται:

* Numpy (version >= 1.21)
* Matplotlib (version >= 3.1)
* Astropy (version >= 5.0)
* Scipy (version >= 1.7)
